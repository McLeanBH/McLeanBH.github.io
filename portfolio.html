  • "Angler's Hook"

    "Angler's Hook"

    Developed using Ember, Angler's Hook is a dynamic weather resource and social media platform for saltwater sport-fishing enthusiasts with an aim to provide automated live-scoring, photo-sharing, and collaboration. For backend database storage, user authentication and relational data, "Angler's Hook" was developed using Parse's REST API and cloud code services. Additional technologies employed include:
    • jQuery • jQuery-ui • PhotoSwipe • Pre-processing libraries Neat and Bourbon • FilePicker.io • Weather Underground REST API for Gelocation, dynamic weather data, and an 8-frame animated radar

  • "Majestic Thai Bistro"

    "Majestic Thai Bistro"

    "Majestic Thai Bistro" is a fully-interactive restaurant website with dynamic menu and e-commerce platform. Developed using Backbone, "Majestic Thai Bistro" allows users to place online orders, specifying deliver or carryout options; provides the restaurant owner with an administrator panel for adding and editing menu items; and includes the implementation of a fully-interactive shopping cart and checkout procedure. Additional technologies implemented include:
    • Extensive JavaScript • jQuery • Underscore.js • Several jQuery animation libraries • The Parse REST API for backend database storage

  • "Surf & Paddle Co."

    "Surf & Paddle Co."

    Surf & Paddle Co." is a pixel-perfect recreation of a standard photoshop comp, developed into a fully-functional blogging platform website. Using SASS and the pre-processing libraries Bourbon and Neat, I implemented a responsive design, with media query breakpoints set for both tablet and mobile devices alike. Requiring the most subtle attention to detail, the final product mirrors the original mockup pixel-for-pixel.
